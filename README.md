# Youtube converter/downloader
Software capaz de baixar converter vídeos do YouTube pela URL. Funções:
- Exibe o canal e o título da URL fornecida pelo usuário:
- Tem a possibilidade de conversão para MP3 ou MP4
- Opção de download
